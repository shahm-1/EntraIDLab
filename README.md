# üåê Entra ID / Azure AD Lab

This lab demonstrates how to manage **Microsoft Entra ID (Azure Active Directory)**, including creating users, assigning licenses, configuring roles, enabling MFA and SSPR, managing groups, and setting conditional access policies. It covers both single-user and bulk operations, as well as inviting external users to collaborate securely.

---

## üìù Lab Overview

The lab walks through:

1. Adding **external members** to an existing group.
2. Assigning **roles** to users individually or via alternative methods.
3. Assigning **licenses** to a single user and to entire groups.
4. Creating users **individually and in bulk via CSV**.
5. Creating and managing **groups**.
6. Configuring **conditional access policies** for specific users.
7. Setting **MFA lockout thresholds** and **password protection policies**.
8. Enabling **MFA** and **self-service password reset (SSPR)** for users/groups.
9. Inviting **external users** to collaborate.

By the end, you have a fully configured Entra ID environment suitable for testing access, roles, licensing, MFA, SSPR, and conditional access policies.
